# archinstall-contrib

Hosted one liner: `curl -sSL https://evoquus.com/s/co -o co && bash co archlinux archinstall` 

You can adapt this to anyone's [fork](https://github.com/archlinux/archinstall/forks?include=active&page=1&period=2y&sort_by=last_updated)

For example for my testing branch ``curl -sSL https://evoquus.com/s/co -o co && bash co h8d13 archinstall-patch testing`
